Sheridan Square Afghan Pattern
Sheridan Square Afghan is a popular crochet pattern that creates a beautiful and intricate afghan or blanket.

MATERIALS:
COATS & CLARK'S "RED HEART" KNITTING WORSTED, 4 Ply (4 oz. "Tangle-Proof" Pull-Out Skeins): 12 skeins of No. 848 Skipper Blue; 2 skeins each of No. 909 Scarlet, No. 227 Canary Yellow and No. 676 Emerald Green.
Milwards Steel Crochet Hook No. 2/0 (double zero). Crochet Hook Size I.
Afghan measures 46 1/2 x 68 inches . .. Each strip measures 51 x 65 1/2 inches.
GAUGE: 7 sc = 2 inches; 4 rows = 1 inch.
FIRST STRIP: (Make 4) Starting at narrow edge with Blue and Size I hook, ch 20.
1st row: Sc in 2nd ch from hook and in each ch across-19 sc. Ch 1, turn.
2nd row: Sc in each sc across.
Drop Blue, attach Yellow. Ch 1, turn.
3rd row: Picking up back loop only throughout, sc in next 3 sc, * make an sc in the sc directly below next sc of next-to-last row-long sc made; skip the sc covered by the long sc, sc in next 3 sc. Repeat from * across. Ch 1, turn.
4th row: Working through both loops, sc in each st across-19 sc.
Break off Yellow, pick up Blue. Ch 1, turn.
5th row: Picking up back loop only throughout, sc in first sc, * long sc in next sc of next-to-last row, skip the sc covered by the long sc, sc in next 3 sc. Repeat from * across, ending with sc in last sc. Ch 1, turn.
6th row: Repeat 4th row.
Drop Blue, attach Scarlet. Ch 1, turn.
7th and 8th rows: Repeat 3rd and 4th rows.
Break off Scarlet, pick up Blue. Ch 1, turn.
9th and 10th rows: Repeat 5th and 4th rows.
Drop Blue, attach Emerald. Ch 1, turn.
11 th and 12th rows: Repeat 3rd and 4th rows.
Break off Emerald, pick up Blue. Ch 1, turn.
13th and 14th rows: Repeat 5th and 4th rows. Ch 1, turn.
15th row: Picking up back loop only throughout, sc in each sc across. Ch 1, turn.
16th row: Repeat 4th row.
17th through 28th row: Repeat 15th and 4th rows alter-nately. Repeat last 26 rows (3rd through 28th row) for pattern. Work in pattern until 10 patterns in all have been completed. Break off and fasten. Strip should measure about 65.5 inches.
SECOND STRIP (Make 4) Work same as for First Strip, working 3rd and 4th rows with Emerald and 11 th and 12th rows with Yellow. Block each strip. Having the end of one strip in line with the beginning of next strip, sew strips together length-wise, alternating patterns.
BORDER 1st rnd: With right side facing, using Blue and No. 2/0 hook, sc closely around entire outer edge, making 3 sc in each corner. Join with sl st to first sc.
Next 5 rnds: Ch 1, picking up back loop only throughout, sc in each sc around, making 3 sc in each corner. Join. At end of last rnd, turn.
Next rnd: (wrong side): Sl st in each sc around. Join, break off and fasten.
